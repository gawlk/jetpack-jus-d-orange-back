# JETPACK DE JUS D'ORANGE - Back end

[![Actions Status](https://github.com/gawlk/jetpack-jus-d-orange-back/workflows/Node%20CI/badge.svg)](https://github.com/gawlk/jetpack-jus-d-orange-back/actions)

## Install

`npm install`

## Start

`npm run start`

## Deamon

### Start deamon

`npm run deamon.start`

### Stop deamon

`npm run deamon.stop`

## Test

### Run test.js

`npm run test.file`

### Unit Tests

`npm run test`

### Unit Tests with watcher

`npm run test.watch`

### Unit Tests with coverage

`npm run test.coverage`

### Mutation tests

`npm run test.mutation`

### API Tests

`npm run test.api`

### All tests

`npm run test.all`
